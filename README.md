# Concert Audio System (CAS)

Профессиональный звуковой микшер для театральных постановок и концертов.

## Установка

1.  Скачайте последнюю версию со страницы [релизов](https://github.com/USERNAME/REPOSITORY/releases) на GitHub.
2.  Распакуйте архив `Concert-Audio-System-v1.0.0-win-x64.zip` в удобное для вас место.
3.  Запустите `Concert Audio System.exe`.

## Основные возможности

*   **Плейлисты**: Управляйте плейлистами из папок на вашем компьютере.
*   **Треки**: Воспроизводите треки в разных режимах (последовательно, зациклить, только текущий).
*   **Звуковые эффекты**: Назначайте звуковые эффекты на 12 пэдов и запускайте их в любой момент.
*   **Микшер**: Регулируйте громкость музыки и эффектов отдельно.
*   **Горячие клавиши**: Управляйте приложением с помощью клавиатуры.
*   **История**: Просматривайте историю воспроизведенных треков.
*   **Таймер**: Используйте таймер обратного отсчета.

## Горячие клавиши

| Клавиша      | Действие                               |
| :----------- | :------------------------------------- |
| **Пробел**   | Пауза / Продолжить воспроизведение     |
| **Esc**      | Остановить все звуки                   |
| **L**        | Блокировка / разблокировка интерфейса |
| **M**        | Mute (паника)                          |
| **←** / **→**| Переключение треков                    |
| **↑** / **↓**| Громкость музыки                       |
| **Shift+↑/↓**| Громкость эффектов                     |
| **1-9, 0, -, =** | Звуковые пады 1-12                     |
